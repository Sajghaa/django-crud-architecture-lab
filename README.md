# Django CRUD Architecture Lab

A structured Django project built to master CRUD operations 
from foundational patterns to production-level architecture.

## Objectives
- Master Django ORM deeply
- Implement CRUD using FBV and CBV
- Apply permissions and authentication
- Implement validation and business logic properly
- Explore performance optimizations
- Build API-based CRUD with DRF

## Architecture Principles
- Clean app separation
- Reusable patterns
- Scalable settings structure
- Production-ready configuration

## Tech Stack
- Django
- PostgreSQL
- Django REST Framework
- Python 3.11+
